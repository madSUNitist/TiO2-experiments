## 1. 开发路线

本项目经历四次重大迭代，每次迭代针对前一个阶段的瓶颈进行定向修复。

### 阶段一 —— 盲目自动寻峰（`scipy.signal.find_peaks`）

初始方案：Savitzky–Golay 平滑 → `find_peaks` → 与 JCPDS #21-1272（锐钛矿）和 #21-1276（金红石）标准峰表做最近邻匹配。

**问题**：`find_peaks` 的参数（prominence、height、distance）对信噪比极度敏感。金红石峰高且宽，占据全谱强度最大值的绝对主导，导致锐钛矿弱峰全部跌破全局阈值。结果是：要么检出数百个噪声伪峰，要么只检出 3–4 个强峰，丢失约 80% 的真实衍射特征。"一刀切"的阈值策略与混合相 XRD 数据天然不兼容——不同相的峰宽和峰高可能相差一个数量级。

### 阶段二 —— 参考引导直接标定

放弃自动寻峰，改为：针对全部 27 个标准参考 2θ 位置（14 个锐钛矿 + 13 个金红石），直接在基线扣除后的谱图上取该位置的强度值，强行放置标志。

**问题**：在理论值位置盲目标记没有物理意义。如果样品不含锐钛矿，25.28° 处的基线噪声值也会被标上 "(101)" 标签。而且参考位置的强度不等于峰强——实际峰位可能由于仪器零点偏移或样品高度位移而偏离理论值 0.2°–0.5°。参考数据应该**引导**搜索，而非**决定**结论。

### 阶段三 —— 双窗口局部峰验证（最终方案）

针对每个参考位置（hkl, 2θ_ref），在预处理后的实际谱图中以小窗口搜索真实局部最大值，再以大窗口计算突出度（prominence）来验证该峰是否物理存在。这是最终采用的生产方案。

### 阶段四 —— 打磨

陆续加入基线扣除、标签碰撞避让（邻近峰的 offset 交替）、透明背景、Spurr–Myers 物相组成估算，以及完整的诊断报告系统。

### 关于相共识投票的放弃

曾测试两轮相共识算法：第一轮标记"争议峰"（另一相在 0.80° 内有重合标准峰），第二轮用无争议峰投票判多数相，再对争议峰重新分配。最终弃用，原因有二：（1）"纯金红石"样品中也可能出现锐钛矿无争议特征峰——投票无法驳倒物理事实；（2）对于本质上无法区分的峰对（如锐钛矿 62.69° vs 金红石 62.74°，仅差 0.05°），强行重新分配制造了虚假的确定性。

## 3. 技术细节

### 3.1 ALS 基线扣除

非对称最小二乘（Asymmetric Least Squares，Eilers & Boelens, *Analytical Chemistry*, 2005）是一种迭代方法，拟合信号下包络的平滑基线，同时抑制峰的贡献。它求解带正则化的加权最小二乘：

```
(W + λ·D) · z = W · y
```

其中 `D = D₂ᵀD₂` 是二阶差分惩罚矩阵（强制基线平滑），`W` 是对角权重矩阵。权重在每次迭代中更新：

```
wᵢ =  p      若 yᵢ > zᵢ   （信号在基线上方 → 低权重，视为峰）
wᵢ = 1 − p  若 yᵢ ≤ zᵢ   （信号在基线处或下方 → 高权重，视为基线）
```

| 参数 | 取值 | 含义 |
|------|------|------|
| λ (lam) | 1 × 10⁶ | 平滑度——越大基线越平坦 |
| p | 0.001 | 不对称性——小值表示仅拟合信号下包络 |
| n_iter | 10 | 迭代次数，对于本数据 10 轮已充分收敛 |

每张图中以 50% 透明度的虚线绘出基线曲线以供审计。但**图谱主线（黑色实线）本身就是基线扣除后的数据**：`smoothed = SavGol(raw − baseline)`。这与专业 XRD 文献的通行做法一致——展示扣除基线后的谱图、同时标出被减去的基线位置。

### 3.2 Savitzky–Golay 平滑

对基线扣除后的数据施加 7 点三次 Savitzky–Golay 滤波。该方法在每个点用其邻域内的三次多项式拟合替代原始值，**保留峰形**（不像移动平均那样削平峰顶、偏移峰位）。小窗口（在 2θ 空间仅 0.07°）提供轻度平滑以压制电子噪声，不扭曲峰形。

### 3.3 双窗口局部峰验证（核心技术）

这是本系统最重要的技术贡献。标准寻峰算法在混合相 XRD 数据上失败的原因：

1. **金红石峰宽且高**（半峰宽约 2.5°）——在全局突出度阈值下压倒一切。
2. **锐钛矿峰窄且矮**——其绝对突出度很小。
3. **固定搜索窗口无法兼顾两者**——窄窗口截断宽峰的真实基底，低估突出度；宽窗口导致相邻参考位置碰撞在同一个局部最大值上。

**解决方案**：将**峰位置搜索窗**与**突出度计算窗**分离。

```
对于每个参考位置 (hkl, 2θ_ref)：

  第一步 —— 窄窗口 (±0.50°)：
    找到局部强度最大值。
    若最大值落在窗口边界则拒绝（边缘伪影）。

  第二步 —— 宽窗口 (±2.0°)：
    向外扩展找到峰的真实左右基底。
    计算 突出度 = 峰高 − max(左基底最小值, 右基底最小值)。
    计算 宽窗极差 = max(窗口内信号) − min(窗口内信号)。
    仅当 突出度 > 宽窗极差 × 0.10 时接受（局部相对阈值）。

  第三步 —— 去重：
    若两个参考位置映射到的峰间距 < 0.15°，
    合并其 hkl 标签（如 "(204)/(002)"），保留较高者。
```

这一方法有两个核心优势：

- **相位无关**：一个小的锐钛矿峰只要在其自身的 ±2.0° 邻域内突出，就通过验证——不管金红石的峰在别处有多高。
- **宽度自适应**：±0.5° 窄窗锁定峰位，保证相邻参考位置不会碰撞出同一个峰；±2.0° 宽窗捕捉全峰基底，即使最宽的金红石峰也能准确评估突出度。

### 3.5 物相定量（Spurr–Myers 公式 + NNLS 全谱分解）

物相组成估算提供两套互补方法：

#### Spurr–Myers 双峰法（1957）

通过两个最强的互不重叠特征峰的强度比估算锐钛矿质量分数：

```
                         100
w_Anatase (%) = ────────────────────────────────
                 1 + 1.26 × I_Rutile(110) / I_Anatase(101)
```

系数 1.26 校正了锐钛矿和金红石对 Cu Kα 辐射的质量吸收系数差异。两个特征峰——锐钛矿 (101) ~25.28° 和金红石 (110) ~27.45°——互不重叠且间距充分，是定量分析的理想选择。

**局限**：仅利用两个峰的信息；若某特征峰区域存在基线波动、噪声或择优取向导致的强度畸变，估算结果将直接受影响。

#### NNLS 全谱内积法（新增）

受傅里叶变换思想的启发——任一信号可表示为一组正交基函数的加权线性组合。在 XRD 定量分析中，实验谱图 `y_exp` 可视为纯相标准谱 `y_Anatase` 与 `y_Rutile` 的线性叠加：

```
y_exp(2θ) ≈ w_A · y_Anatase(2θ) + w_R · y_Rutile(2θ)
```

求解思路：
1. 从 CIF 晶体结构出发，用 pymatgen 计算两相的理想衍射峰（棒状谱）
2. 以高斯函数（FWHM = 1.0°）对棒状谱卷积展宽，生成连续参考谱，插值到与实验数据相同的 2θ 网格
3. 将实验谱与参考谱均归一化至最大强度为 1
4. 使用非负最小二乘（NNLS, Lawson & Hanson, 1974）求解权重 `(w_A, w_R)`，使得重构误差 `∥y_exp − (w_A·y_A + w_R·y_R)∥²` 在 `w ≥ 0` 约束下最小化
5. 物相比例由权重归一化得到：`f_Anatase = w_A / (w_A + w_R)`

**优势**：
- **利用全部衍射信息**：不依赖单一特征峰，对局部噪声和择优取向更具鲁棒性
- **无需经验常数**：不引入 Spurr–Myers 公式中的 RIR 系数 1.26
- **自带拟合质量评估**：R² 值直接反映两相线性叠加模型对实验数据的解释程度。当 R² 较低时，提示可能存在非晶相、第三相，或晶粒尺寸/择优取向导致的峰形偏离
- **数学等价于傅里叶域内的匹配滤波**：内积 ⟨y_exp, y_phase⟩ 本质上是将实验谱投影到各标准相的子空间上，寻找最佳线性组合。这与傅里叶变换将时域信号投影到正弦基上的思想一致——只是这里的"基函数"不是正弦波，而是各纯相的衍射谱

**当前参数**：FWHM = 1.0°，2θ 计算范围 20°–65°（排除低角与高角噪声区）。

两种方法的结果同时标注于每张图谱左上角，便于交叉验证。

### 3.6 标签碰撞避让

当两个验证通过的峰在 2θ 方向上间距小于 1.5° 时，其 hkl 标注会垂直重叠。本系统采用**确定性交替策略**：相邻峰在标准偏移（+8 pt）和加高偏移（+16 pt）之间交替，形成 `正常↗抬高↗正常↗抬高` 的格局，所有标签均朝上、无一下沉到数据点以下。这避免了基于优化的碰撞求解器（如 `adjustText`）在密集 XRD 图上产生的混乱结果。

## 5. 技术亮点

1. **双窗口突出度验证**。窄窗/宽窗的分离解决了位置精度（小窗区分相邻峰）与突出度精度（大窗捕捉全峰基底）之间的根本冲突。这种参考数据库引导的局部验证方法对 XRD 峰标记而言是新颖的。

2. **局部相对阈值**。突出度判定以局部信号极差（`wide_range`）为基准，而非全局最大强度。这使得峰过滤器天然具有相位不变性——一个小的锐钛矿峰只要在自己的邻域内显著突出即可通过，与远处金红石大峰的高度全然无关。

3. **ALS 基线透明展示**。基线曲线以半透明虚线形式保留在每张图中，使扣除过程完全可审计。商业软件（Jade/HighScore）通常用 Chebyshev 多项式 + 手动锚点的方式，可复现性不如 ALS——相同的参数、相同的输入，ALS 每次都给出相同结果。

4. **确定性标签交错**。与基于优化的文本放置算法（可能在密集图上产生混乱结果）不同，简单的交替偏移策略保证了可预测、可读的标签位置，且零额外计算量。

5. **参考引导而非参考独断**。峰标记于**实测** 2θ 位置，而非理论值。这尊重了仪器偏移的物理现实，同时保持了与 JCPDS 数据库的完全可追溯性。

6. **NNLS 全谱物相分解**。受傅里叶变换启发，将实验 XRD 谱图投影到纯相参考谱张成的子空间上，通过非负最小二乘求解最佳线性组合。相比仅依赖两个峰的 Spurr–Myers 公式，全谱方法利用全部衍射信息，无需经验常数，自带 R² 拟合优度评估，且对单峰噪声和择优取向具有天然的鲁棒性。

## 7. 配置与可复现性

全部处理管线由 `plot_xrd.py` 中的一个 `CONFIG` 字典控制：

| 参数 | 当前值 | 作用 |
|------|--------|------|
| `baseline_lam` | 1 × 10⁶ | ALS 平滑度 |
| `baseline_p` | 0.001 | ALS 非对称度 |
| `baseline_niter` | 10 | ALS 迭代次数 |
| `sg_window` | 7 | SavGol 窗口（点数） |
| `sg_polyorder` | 3 | SavGol 多项式阶数 |
| `local_search_window` | 0.50° | 窄窗——峰位搜索 |
| `local_base_window` | 2.0° | 宽窗——突出度计算 |
| `local_prominence_min` | 0.10 | 局部相对阈值 |
| `dedup_tolerance` | 0.15° | 相同峰去重容差 |
| `nnls_fwhm` | 1.0° | NNLS 高斯展宽半峰宽 |
| `nnls_tt_mask` | (20°, 65°) | NNLS 计算 2θ 范围 |

**可复现性**：所有算法均为确定性算法。给定相同的原始数据，每次必然产生完全相同的图谱。无需人工干预。

**运行命令**：
```bash
uv run python plot_xrd.py                      # 批量绘制全部样品
uv run python plot_xrd.py --diagnose sample_01 # 单样品完整诊断
```
